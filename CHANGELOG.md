# Changelog

### Added

- Initial public release of Codex Quota Monitor.
- Popup view for current 5-hour session usage, weekly usage, reset timers, plan type, and remaining Codex credits.

### Improved

- Distinct orange visual identity for the Codex extension.
- Clearer quota labels for the active 5-hour session and weekly limits.

### Fixed

- Refresh handling when no analytics tab is already open.
- Light-theme contrast for the limit-reached state.
